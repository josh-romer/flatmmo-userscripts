Flatmmo and flatmmo wiki userscripts

## Scripts

### table-links.js
Adds anchor links to wikitable rows based on the first column text. Each row gets a clickable '#' link that creates a direct URL to that specific row.

### header-links.js
Adds anchor links to all headers (h1-h6) on wiki pages. Each header gets a clickable '#' link for easy sharing and navigation.

### image-links.js
Shows a modal on image hover that displays which other wiki pages use that image. Helps navigate between related pages through shared images.

### draw-player-on-top.js
Modifies player draw function to always draw the user's player sprite last (on top).
